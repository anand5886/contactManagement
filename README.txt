**Client Application**
1. CONTACT-APP developed using react.js
2. Folder src/components/ has the components like Login, Home, Contacts, SharedServices
3. src/components/SharedServices has all generic(reusable) components
4. src/components/SharedServices/Services.js file has the get and post methods to interact with API's.

**How to run the app**
1. Application can be ran using Visual Studio Code.
2. Open terminal and run npm start.
